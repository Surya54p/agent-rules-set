# agent-rules-set

A repository containing a personal rule set for AI agents.

This project stores guidance and conventions used when building, reviewing, and coordinating AI agent behavior across development roles.

## Purpose
- Capture agent coordination rules.
- Document workflow constraints.
- Keep a shared reference for contributors.

## Contents
- `agents.md` – core agent rules and responsibilities.
- `coordination.md` – collaboration log and task tracking.

## Usage
Use this repository as a reference when designing or reviewing AI assistant workflows and team coordination.
